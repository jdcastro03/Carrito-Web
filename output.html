Practica 3 HTML5 Inputs

Output Element

Esta etiqueta se utiliza para mostrar resultados en una página web. Imagina que estás usando una calculadora en línea y quieres ver el resultado de una operación. La etiqueta te permite mostrar ese resultado en la página.

Input Type Password

La etiqueta Input con el atributo type="password" se utiliza en formularios HTML para crear un campo de entrada de contraseña. Cuando el usuario ingresa datos en este campo, los caracteres se muestran como asteriscos o puntos para ocultar la contraseña y proteger la privacidad del usuario.

Input Type Reset

La etiqueta Input con el atributo type="reset" se utiliza en formularios HTML para crear un botón de reinicio. Cuando el usuario hace clic en este botón dentro de un formulario, se restablecen todos los campos del formulario a sus valores predeterminados o vacíos.
